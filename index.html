    📖 Manuel d'utilisation - DessNotes

    🚀 Démarrage rapide

    DessNotes est un éditeur d'annotation d'images. Idéal pour prendre des notes visuelles sur des photos (mesures, repères, croquis...).

    1. Cliquez sur 📂 Ouvrir ou glissez une image dans la fenêtre.
    2. Cliquez sur ➕ pour ajouter un calque transparent par-dessus.
    3. Choisissez un outil (pinceau, ligne, texte...) et annotez.
    4. Exportez en 🖼️ PNG ou 🖨️ Imprimez.

    🧰 Outils

    • ↔️ Déplacer (V) — Sélectionnez un calque dans le panneau de droite, puis cliquez-glissez sur le canvas pour le déplacer. Utilisez les poignées aux coins pour redimensionner. La poignée du haut sert à pivoter.
    • 🖌️ Pinceau (B) — Dessine à main levée. Réglez la taille, la couleur et l'opacité dans le panneau de droite. Choisissez d'abord le calque cible !
    • 📏 Ligne (L) — Tracez des lignes droites :
      1. Premier clic = point de départ
      2. Bougez la souris : un aperçu gris suit le curseur
      3. Deuxième clic = valide la ligne définitive
      4. Échap ou clic droit = annule
      Astuce : maintenez Maj pour contraindre à 0°, 45° ou 90°.
    • 🧽 Gomme (E) — Efface sur le calque actif. Utilisez la même taille que le pinceau.
    • 🅣 Texte (T) — Important, lisez bien :
      1. Sélectionnez l'outil 🅣 Texte.
      2. Cliquez sur le canvas à l'endroit où placer le texte. Une zone est créée avec un texte par défaut.
      3. Pour modifier le contenu : repassez en outil ↔️ Déplacer (V), puis double-cliquez sur la zone de texte. Saisissez votre texte et cliquez ailleurs pour valider.
      4. Pour déplacer ou redimensionner la zone : outil Déplacer + glissez le calque ou ses poignées.
      5. La couleur et la taille du texte se règlent dans le panneau de droite avant la création (ou se modifient ensuite via les contrôles dédiés).
    • 💧 Pipette (I) — Cliquez sur une couleur de l'image pour la définir comme couleur active du pinceau.
    • ✂️ Recadrer (C) — Tracez un rectangle sur l'image, puis validez avec ✓ ou annulez avec ✗. Recadre tout le document.

    🗂️ Calques

    • ➕ Ajoute un calque transparent vide.
    • 🖼️ Ajoute une nouvelle image en tant que calque.
    • 👁️ Affiche/masque un calque (clic sur l'œil).
    • Double-clic sur le nom d'un calque pour le renommer.
    • ⬆️ ⬇️ Réordonner le calque sélectionné.
    • 📑 Dupliquer le calque.
    • 🗑️ Supprimer le calque sélectionné (ou touche Suppr).
    • Opacité : curseur en bas du panneau, agit sur le calque sélectionné.

    🔍 Navigation & Zoom

    • Molette de souris = zoom/dézoom centré sur le curseur.
    • Boutons 🔍+ / 🔍- = zoom par paliers.
    • Bouton ⤢ = ajuste à la fenêtre.

    💾 Sauvegarde & Export

    • 💾 Sauver = enregistre un fichier projet .dnotes (réouvrable plus tard avec tous les calques).
    • 📁 Projet = recharge un fichier .dnotes.
    • 🖼️ Export PNG = exporte l'image finale aplatie (transparence préservée).
    • 🖨️ Imprimer = ouvre la boîte de dialogue d'impression.
    • Auto-sauvegarde : DessNotes sauve automatiquement votre travail toutes les 30 s. Si vous fermez par accident, vous serez invité à restaurer au prochain démarrage.

    💡 Cas d'usage type : annoter une photo

    Exemple : indiquer la hauteur d'une haie sur une photo.

    1. 📂 Ouvrir la photo de la haie.
    2. ➕ Ajouter un calque "Trait actuel".
    3. 📏 Outil ligne, couleur cyan, tracer un trait vertical de la base au sommet.
    4. ➕ Ajouter un calque "Texte actuel".
    5. 🅣 Outil texte, cliquer à côté → revenir en V → double-clic → écrire "2m80".
    6. Répéter avec un autre calque en vert pour la hauteur souhaitée "2m".
    7. 🖼️ Exporter en PNG, ou 💾 sauver le projet pour modifier plus tard.

    📱 Installation (PWA)

    DessNotes peut être installé comme une application : sur Chrome/Edge, cliquez sur le bouton ⬇️ Installer dans la barre supérieure (s'affiche quand l'installation est possible), ou utilisez le menu du navigateur. Sur smartphone : "Ajouter à l'écran d'accueil". Une fois installée, l'app fonctionne hors ligne.
